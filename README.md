# Bola na Rede

Front-end de uma PoC que valida todo o fluxo de autenticação de usuários, com cadastro, login, confirmação em duas etapas e rotas de dashboard protegidas.

## Blibliotecas usadas:
- React Router
- React icons
- TailwindCSS
- Framer Motion
- Swiper
- Axios

## Instalação
1. Clone o repositório:
   ```bash
    git clone https://github.com/Mateus-Ca13/Bola_na_Rede.git
   ```
2. Navegue até o diretório do projeto:
   ```bash
   cd Bola_na_Rede
   ```
3. Instale as dependências:
   ```bash
   npm install
    ```
4. Inicie o servidor de desenvolvimento:
   ```bash
   npm run dev
    ```
